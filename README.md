# 🏥 Hospital Management System (SQL Project)

## 📌 Project Overview

This project is a Hospital Management System database built using MySQL. It simulates a real-world hospital environment by managing patients, doctors, appointments, treatments, and billing information.

The project demonstrates SQL skills by answering business-related questions using joins, aggregate functions, grouping, subqueries, and filtering.

---

## 🛠️ Tools Used

* MySQL
* MySQL Workbench
* GitHub

---

## 🗄️ Database Tables

* Patients
* Doctors
* Appointments
* Treatments
* Bills

---

## 📊 Business Questions Solved

* Total number of patients
* Patients from each city
* Oldest patient
* Number of appointments by doctor
* Busiest day of the week
* Most common disease
* Most common treatment
* Most expensive treatment
* Revenue by department
* Total hospital revenue
* Number of patients treated by each doctor
* Average consultation fee by department
* Paid vs Pending bills
* Top revenue-generating doctor
* Highest billing patient
